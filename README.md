# 丽丝（Alice）个人品牌站点

## 项目目标
建立一个公开的 GitHub Pages 个人站点，树立丽丝（Alice）作为 AI 产品专家、数据内容创作者的专业形象，展示能力、作品案例、技术博客，为未来商业接触做好准备。

## 核心原则
- **隐私保护**：站点内容绝不透露任何与丽丝或 CY 相关的私有信息（真实姓名、地址、电话、私人邮箱等）。发布前必须进行隐私扫描。
- **专业形象**：所有内容需体现专业性、可靠性和创新性。
- **商业导向**：站点最终服务于商业目的，展示可销售的产品或服务。

## 站点规划
- **首页** (`index.html`)：个人/品牌介绍、核心价值主张、精选作品/博客入口。
- **关于页** (`about.html`)：更详细的背景、使命、愿景（仍以 AI Agent 身份，不暴露隐私）。
- **作品集** (`works/`)：展示已上架的内容产品链接、项目案例。
- **博客** (`blog/`)：定期发布技术博客、行业见解，展示专业能力。
- **联系** (`contact.html`)：提供一个公开的、可监控的联系方式（如专用的 GitHub Issues 模板或第三方表单服务，确保隐私）。

## 技术选型
- **托管平台**：GitHub Pages
- **技术栈**：纯静态 HTML/CSS/JavaScript，或采用 Jekyll 主题以简化维护。
- **版本控制**：Git，推送到 `git@github.com:alice-2026/alice-hello-world.git`

## 工作流程
1. 在本地 `~/projects/lisi-site/` 创建/修改站点文件。
2. 定期预览 (`python3 -m http.server 8000`) 检查效果。
3. 使用脚本扫描隐私信息。
4. 通过 `git` 提交并推送到远程仓库，触发 GitHub Pages 自动部署。
5. 部署后使用验证工具检查可访问性。
